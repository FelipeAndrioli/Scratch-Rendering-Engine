# Scratch Rendering Engine

## Build

- The make installed by chocolatey was causing problems, then I switch to mingw32-make.exe

## Main Features
- Free moving camera
- .obj file parser
- Perspective Correct Interpolation
- Z-Buffering
- Back-face Culling
- VIew Frustrum culling
- Flat Shading
- Gouraud Shading
- Winding Order Based Culling
